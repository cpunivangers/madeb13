Installation Windows 10 dans une Machine Virtuelle (VM)

Télécharger l’image iso de Windows 10 :

https://www.microsoft.com/fr-fr/software-download/windows10ISO

Affichage de la licence Windows inclus dans le Bios 

sudo tail -c +56 /sys/firmware/acpi/tables/MSDM

Cette licence vous servira à activer votre Windows dans la VM

Faire l’installation dans une VM

Vous pouvez utiliser Oracle Virtualbox 6.1 ou directement pour Linux QEMU-KVM ( https://blog.tremplin.ens-lyon.fr/GerardVidal/qemuKVM-Win10.html )

Donner 4 Go de mémoire et 30 Go de disque minimum pour votre VM.

“ Insérer ” l’image votre image iso de Windows dans la lecteur CD de la machine virtuelle et lancer l’installation de votre Windows.

N’oubliez pas les additions invités pour virtualbox ou les virtio-win pour qemu-kvm.

Une fois la machine Windows10 totalement opérationnelle, vous pouvez ajouter les logiciels que vous souhaitez.

Si votre machine Windows s’appelle RDPWindows vous pourrez tester WinApps for Linux ( https://github.com/Fmstrat/winapps ) pour intégrer vos applications Windows dans votre Linux.**
